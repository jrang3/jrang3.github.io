Othello Recommender Web App

Full-stack web application that detects an Othello board from an image, computes optimal moves using a Minimax algorithm, and visually overlays suggestions, while logging past submissions to a SQLite database.

üîç Overview

This project is a full-stack extension of my original Othello board recognition project developed for the Computational Photography (CS 445) course. While the original class project focused on detecting and classifying piece positions on an Othello board using various computer vision techniques, this web app builds on that foundation by adding end-to-end functionality, AI move prediction, and user interactivity.

Users can upload an image of a real-world Othello board. The backend uses OpenCV and piece classification logic to detect all positions, then runs a Minimax algorithm with alpha-beta pruning to compute the optimal move for both players. It also overlays the computed moves directly onto the uploaded image using OpenCV. All submissions are stored in a SQLite database, enabling users to revisit previous results through a persistent history feature.

The results are rendered and displayed through a modern React-based frontend, which provides responsive design, intuitive controls, and access to prior submissions.
