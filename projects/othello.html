Othello Board Recognition Project

Developed an image processing pipeline to detect and score Othello game boards from photos using OpenCV, achieving 95.62% classification accuracy.

üîç Overview

While many research efforts focus on recognizing board states in games like chess, Othello remains relatively underexplored. For my final project in Computational Photography (CS 445), I collaborated with two classmates to develop an Othello Board Recognition system that extracts the board state and calculates the game score from a single input image.

We approached this in two stages. In Stage 1, we used OpenCV to localize the board using three techniques ‚Äî Naive Chroma Keying, Fast Line Detection, and the Hough Transform. In Stage 2, we applied Gaussian blurring and homography to classify each cell as black, white, or empty. After classification, we computed the board score using a scoring scheme where white = 1, black = -1, and empty = 0. Our Hough-based pipeline achieved a board-wide classification accuracy of 95.62%.
