Smart Email Reply — Gmail Add-on

Gmail add-on that generates personalized AI-powered email replies using thread context, tone preferences, and custom user instructions.

🔍 Overview

Composing effective email replies can be time-consuming, especially when dealing with long threads or the need to match specific tones. To address this, I collaborated with two teammates to develop Smart Email Reply, a Gmail add-on that uses Google’s Gemini API to generate intelligent, context-aware draft responses.

I developed prompt assembly and thread extraction algorithms that consolidate full conversation history into structured prompts, enabling the model to generate more personalized and relevant replies. The add-on allows users to select tone and language, add optional instructions (e.g., “Keep it concise” or “Reference Section 4”), and preserves attachments while integrating directly into Gmail via Google Apps Script. In informal testing with 10 users, 90% rated the responses as accurate and contextually appropriate.

⚙️ Tools & Technologies Used
